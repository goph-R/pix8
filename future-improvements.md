# Future Improvements

## Palette Handling
Full-fledged palette management: rearrange, sort, swap, import/export palettes, palette presets, color ramps.

## Fill Tool
Flood fill with configurable tolerance for indexed color workflows.

## Gradient
Indexed gradient tool that maps a range of palette indices across a line or shape.

## Indexed Opacity
Per-pixel opacity using palette index remapping rather than true alpha, staying faithful to the 256-color indexed model.

## Mask Effect
The core vision of the project. Layer mask effects driven by palette indices, enabling VGA-era compositing and visual effects within the indexed color paradigm.
